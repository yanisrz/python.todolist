# python.todolist 


Ce projet est une to do list (liste de taches) servant de demonstration. 

Le fonctionnement est simple :
      Plusieurs options sont proposées: 
         - Voir la liste de taches; 
         - Ajouter une tache a cette liste; 
         - Suprimer une tache de cette liste;
         - Suprimer toute les taches de cette liste;
         - Quitter le programme.

      Pour chaques options, un chiffre y est respectivement lié:  1; 2; 3; 4; 5.
      Pour valider chaque actons il faut appuyer sur la touche "entré".
